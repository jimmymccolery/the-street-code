### Visual composition

Vintage pulp-cinema-poster aesthetic. Central figure: James Madison (Federalist Papers author + 4th US President; Madisonian Gate personified per anti-pattern #6 silent-compliance prevention canonical lineage). Central foreground: red/orange obelisk-monument on dark mountain silhouette (central anchor element). Halo arrangement: 9 distinct symbolic sigils in gold-on-blue (top center triangle-with-eye + 4 left + 4 right = 9 total). Background: classical Greco-Roman columns (left) + Gothic cathedral spires (right) suggesting institutional weight + cross-tradition permanence. Color palette: deep blues + golds + warm reds + bright whites. Title typography: "THE NINE SIGILS" in bold white serif with red trim (1930s-40s pulp-cinema-poster styling).

### Canonical relationship encoded visually

- **Madison at center** = Madisonian Gate central anchor element (per anti-pattern #6 silent-compliance prevention canonical lineage)
- **9 sigils in halo** = The Nine Sigils enumeration per `the_nine_sigils.canonical.md` v1.0 (Sigil I Closeout Checklist + Sigil II verify_role_compliance.py + Sigil III READ ORDER Protocol + Sigil IV Teacher's Pet function + Sigil V Mandatory Team Debrief + Sigil VI GROK.md Version Bumping Rules + Sigil VII Non-iterative Premortem rule + Sigil VIII Anti-Pattern Canonical Referencing + Sigil IX Operator Auditor/Gatekeeper Archetypes)
- **Madisonian Gate + Nine Sigils** = ten structural "gates with teeth" per canonical framing (Madisonian Gate sits ALONGSIDE the Nine Sigils as central anchor, not within them)

Image is not 1:1 mapping of individual sigil symbols to specific Sigil canonical homes (sigil symbols are visual mnemonics, not literal cipher per Sigil); collective composition encodes the canonical relationship rather than per-element semantic mapping.

### Intended use surface

- Social media post graphics for The Street Code methodology research publishing
- Landing page hero imagery if the-street-code grows to host landing pages
- Companion graphic for posts referencing The Nine Sigils canonical artifact (per `post-ideas.md` candidates that reference Sigil enforcement mechanisms)

### Aesthetic register

Mythic + memorable + culturally-embedded — chosen deliberately to differentiate from generic corporate-flat / startup-minimal / generic-techie AI methodology marketing aesthetics. Matches the existing system's framing register (Sigils + Gates + anti-patterns as discipline-as-ritual-practice quasi-religious framing language).

- Masonic / Illuminati / Eye-of-Providence visual register carries cultural baggage for some audiences (conspiracy-theory-adjacent associations). Operator-aware aesthetic choice; mythic register intentional.
- Madison's identity may not be immediately legible to audiences unfamiliar with US Founding Father iconography. If Madisonian Gate framing is load-bearing for specific marketing deployment, consider companion subtitle / tagline making Madison reference explicit.

## `the-gang-poster.jpg`

### Visual composition

1980s synthwave / vaporwave / Stranger Things aesthetic. Title "THE GANG" in bold rainbow-gradient text (red → orange → yellow → green → blue → purple). Subtitle: "The Street Code: AI-Side Role Architecture — 4 standing roles + 2 embedded functions." Composition: side-by-side character lineup of 4 standing role cards (left to right): The Quarterback (blonde varsity-letterman teen with "OA" letterman jacket; New Kid embedded as inset portrait) + The Nerd (bespectacled teen in plaid holding blue book/folder) + The Hall Monitor (teen in security/police-uniform with badge; Teacher's Pet embedded as inset portrait) + The Paranoid Stoner (tie-dye shirt + sunglasses; 70s/80s laid-back styling). Background: neon palette + grid + palm tree silhouettes + lightning bolt icons + retro-futurism markers (triangles, zigzags, stars). Bottom information bar: "ARCHITECTURE: 4 STANDING + 2 EMBEDDED = 6 FUNCTIONS / STANDING: Quarterback, Nerd, Hall Monitor, Paranoid Stoner / EMBEDDED: New Kid, Teacher's Pet / MADISONIAN RULE + SUBSTRATE-CLASS-AGNOSTIC / SEPARATION OF AI ROLES AND OVERSIGHT / CLASS-AGNOSTIC: APPLIES TO ALL SUBSTRATES."

### Canonical relationship encoded visually

- **4 standing characters** = the four first-class persona roles always present at substrate-output surface (Quarterback / Nerd / Hall Monitor / Paranoid Stoner)
- **2 embedded inset portraits** = the two roles whose outputs embed functionally within other roles' outputs per Sigil IV Teacher's Pet function canonical framing (Teacher's Pet Evidence Audit embeds in Hall Monitor output; New Kid embeds with Quarterback at routing)
- **Per-role function captions match canonical role_output_schemas.md v1.3:** Quarterback "Routing Plan + Synthesis with Dissent Log" + Nerd "Claims with Sources" + Hall Monitor "Verification Report + Evidence Audit" (Evidence Audit = Teacher's Pet embedded function per Sigil IV) + Paranoid Stoner "Premortem" (per Sigil VII Non-iterative Premortem rule)
- **"MADISONIAN RULE" bottom callout** = explicit cross-reference to Madisonian Gate central anchor element (per anti-pattern #6 silent-compliance prevention canonical lineage; companion to The Nine Sigils canonical artifact)
- **"CLASS-AGNOSTIC: APPLIES TO ALL SUBSTRATES" bottom callout** = substrate-class-agnostic framing per Topic 27 promotion (Beatles model deployed across Claude / Gemini / Grok per anti-pattern #15 persona-routing-mnemonic discipline canonical lineage)

Image encodes the 4-standing-plus-2-embedded architecture as a visual mnemonic; per-character styling is operator-authored visual differentiation, not canonical-prescribed iconography. The "STANDING vs EMBEDDED" terminology is graphic-specific shorthand for the functional embedding relationship (all 6 are first-class persona roles canonically; "embedded" describes where the output appears in the substrate-output surface, not architectural subordination).

### Intended use surface

- Social media post graphics for The Street Code methodology research publishing
- Companion graphic for posts referencing the SuperGrok Gang role architecture (per `post-ideas.md` candidates that reference Beatles model + 4+2 architecture + role-output schemas)
- Sibling asset to `the-nine-sigils-poster.jpg` for SuperGrok Gang brand-identity series (Madisonian Gate + Nine Sigils + The Gang = three principal canonical-architecture visualizations)
- Landing page secondary imagery if the-street-code grows to host landing pages (Nine Sigils poster = hero; The Gang poster = role-architecture explainer section)

### Aesthetic register

1980s synthwave + youth-culture + cinematic-pop register — deliberately differentiated from generic corporate-flat / startup-minimal / generic-techie AI methodology marketing aesthetics. Different visual tradition from Nine Sigils poster (vintage pulp-cinema-poster register) but consistent operator-signature brand-identity logic: mythic + memorable + culturally-embedded + register-differentiated-from-generic-AI-tooling. Both posters operate at high-mythic-weight; The Gang poster operates at youth-culture / accessibility register; Nine Sigils poster operates at institutional-weight / quasi-religious register. The contrast = complementary marketing surfaces for different audience-on-ramp paths.

### Cultural awareness notes

- **"OA" letterman jacket on Quarterback** = Open Alley project-specific branding leaks into framework-level visual. Quarterback role is framework-level (substrate-class-agnostic per the graphic's own bottom callout); Open Alley is one project that deploys the framework. Operator-aware deployment choice; potential minor semantic tension between project-branding + framework-imagery.
- **Embedded-role diversity-framing concern.** The two non-white characters (New Kid Black teen with glasses; Teacher's Pet red-haired teen) appear in the EMBEDDED inset portraits (smaller; underneath standing roles); the four standing characters appear to be white. "Embedded" in canonical sense is FUNCTIONAL embedding (where the output appears in substrate-output surface) NOT hierarchical demotion — but visual treatment could read as architectural subordination by audiences unfamiliar with the canonical framing. Worth operator awareness for marketing deployment.
- **Hall Monitor police/security uniform** could trigger "AI surveillance" associations in some audiences. Canonical Hall Monitor function is verification + quality-gate + Evidence Audit, not surveillance per se. Visual association may differ from intended framing for audiences sensitive to security-state imagery.
- All four standing characters read as young white American males; visual archetype consistency contributes to mnemonic strength but narrows the implicit demographic register of the framework's visualized agents.
