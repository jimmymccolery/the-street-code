### Normalized fixture schema

`NPC(personality)` per-NPC personality descriptor including trait routing plus initial affective baseline plus initial cultural-baseline dispatch key.

Ordered pair sentiment `S(A→B)` scalar in [-1, +1] encoding A's affective disposition toward B at trace start.

Layer 3 accumulator tuple `L3 = VF/VS/AF/AS/SF/SS` where V=valence, A=arousal, S=stress; F/S=fast/slow channel.

Event affect marker `ΔV/ΔA/ΔS` per Layer 1 event encoding fast-channel affective deltas.

Endpoint answer key `expected_action` per trace specifying the endpoint action match criterion.

Required ablation conditions per trace specifying which mechanism-ablation runs the trace requires for pass.

Mechanism cross-check per trace specifying which V1 architectural mechanism the trace exercises (pair-summary consolidation, slow consolidation, trait routing, communal-ledger, sparse coding H4, cultural baseline H5).

## Substrate C promotion-forcing plus clogging-pressure required-ablation conditions

Substrate C's DF-derived promotion-forcing plus clogging-pressure conditions extend the required-ablation surface at T01 plus T02 plus T07. The conditions test Dwarf Fortress emotional-memory-tier promotion bias plus long-term-slot clogging failure modes at the framework's V1 schema surface.

**Promotion-forcing condition:** the trace runner injects high-volume low-salience positive events during Y1-Y2 sufficient to fill 8 long-term slots with positive memories, then runs the base trace event sequence. Pass criterion: strong negative or high-salience events during Y3-Y4 must still promote to Layer 2 despite the pre-filled slot pressure. Failure mode exposed: slot clogging preventing stronger relevant memories from entering long-term.

**Clogging-pressure condition:** the trace runner runs the base trace event sequence but injects same-category repeated positive events at Y2-Y3 sufficient to trigger category-grouping-based redundancy elimination (only strongest emotion per category survives). Pass criterion: nuanced-relational-history events must still promote despite same-category redundancy pressure. Failure mode exposed: category grouping limiting nuance where subtle cumulative relational history is lost.

The Substrate C promotion-forcing plus clogging-pressure conditions execute alongside the trace's standard required ablations. The conditions are canonical requirements at T01 plus T02 plus T07; other traces may adopt the conditions as optional at operator discretion.
